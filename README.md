#DRIVER_board

This repository contains hardware design of six motor control driver.

SPECIFICATIONS:

- board size 11*15cm
- 12 BTS MOSFET on board
- 2 snh4 MOSFET driver
- operating voltage 12v to 37v
- Current limitation level of 43 A typ.
